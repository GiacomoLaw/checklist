C1 checklist

For A320 series, and Boeing 737 BBJ.

Pre-start

Briefing
Brakes - On
Throttle - Idle
Spoilers - Off
Flaps - Retracted
Seatbelts - On
No Smoking - On
Nav lights - Off
Beacon lights - On
Landing lights - Off
Strobe lights - Off
Flight plan - Filed
Autopilot - Set
Flight Controls - Tested

After Start

Seatbelts - On
No Smoking - On
Nav lights - On
Takeoff flaps - Set
Brakes - Off

Hold Short/Before Takeoff

Briefing - Done
Landing lights - On
Strobe lights - On
Takeoff flaps - Checked
Flight controls - Tested

Climboout

Gear - Retracted
Flaps - Retracted
Autopilot - Engage
Landing lights - Off (+FL100)
Seatbelts - Off (+FL100)
No Smoking - On (+FL100)

Approach

Seatbelts - On
No Smoking - On
Landing lights - On
APPR - Set if needed
Flaps - Set
Gear - Down and locked
Spoilers - Armed
Autobrake - Set
Trim - Set
