C2 checklist

- Cessna 172/208, Cirrus SR22


Pre-start

Briefing
Brakes - On
Throttle - Idle
Flaps - Retracted
Nav lights - Off
Beacon lights - On
Landing lights - Off
Strobe lights - Off
Flight plan - Filed
Autopilot - Set
Flight Controls - Tested

After Start

Nav lights - On
Takeoff flaps - Set
Brakes - Off

Hold Short/Before Takeoff

Briefing - Done
Landing lights - On
Strobe lights - On
Takeoff flaps - Checked
Flight controls - Tested

Climboout

Flaps - Retracted
Autopilot - Engage
Landing lights - Off (+FL100)

Approach

Landing lights - On
APPR - Set if needed
Flaps - Set
Autobrake - Set
Trim - Set

After-landing

Flaps - Retracted
Landing lights - Off
Strobe lights - Off

Parking

Brakes - On
Throttle - Idle

Shutdown

Brakes - On
Throttle - Idle
Flaps - Retracted
Spoilers - Off
Landing lights - Off
Strobe lights - Off Nav lights - Off
Beacon lights - On
Trim - None
