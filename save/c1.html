C1 checklist

- For A320/30/40/80, Boeing 717/37/47/57/67/77/87/C-17, CRJ 200, Dash 8, Citation X and ERJ series

Pre-start

Briefing
Brakes - On
Throttle - Idle
Spoilers - Off
Flaps - Retracted
Seatbelts - On
No Smoking - On
Nav lights - Off
Beacon lights - On
Landing lights - Off
Strobe lights - Off
Flight plan - Filed
Autopilot - Set
Flight Controls - Tested

After Start

Seatbelts - On
No Smoking - On
Nav lights - On
Takeoff flaps - Set
Brakes - Off

Hold Short/Before Takeoff

Briefing - Done
Landing lights - On
Strobe lights - On
Takeoff flaps - Checked
Flight controls - Tested

Climbout

Gear - Retracted
Flaps - Retracted
Autopilot - Engage
Landing lights - Off (+FL100)
Seatbelts - Off (+FL100)
No Smoking - On (+FL100)

Approach

Seatbelts - On
No Smoking - On
Landing lights - On
APPR - Set if needed
Flaps - Set
Gear - Down and locked
Spoilers - Armed
Autobrake - Set
Trim - Set

After-landing

Spoilers - Off
Flaps - Retracted
Landing lights - Off
Strobe lights - Off
(APPR - Disengaged)

Parking

Brakes - On
Throttle - Idle
Seatbelts - Off
No Smoking - On

Shutdown

Brakes - On
Throttle - Idle
Flaps - Retracted
Spoilers - Off
Landing lights - Off
Strobe lights - Off
Nav lights - Off
Beacon lights - On
Trim - None
